**Framework:** Flutter  
**Dependencies:** http, flutter_svg, 


## Run Locally

Clone the project

```bash
  git clone https://github.com/subrotokumar/cryptobook.git
```

Go to the project directory

```bash
  cd cryptobook
```

Install dependencies

```bash
  flutter pub get
```

Start the server

```bash
  flutter run
```
